# LTchatV2
Simple Webchat Python<br>
# How to<br>
Clone repository
`git clone https://github.com/Linux-Scratcher/LTchatV2.git`<br>
Go in the folder
`cd LTchatV2`<br>
Install flask & socketio
`sudo apt install flask`<br>
`pip3 install socketio`<br>
Run
`python3 app.py`<br>
Go to url
`http://localhost:5006/`
